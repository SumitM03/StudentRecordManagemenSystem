# Student Record Management System

## Overview
The Student Record Management System is a Java-based console application developed to manage student records efficiently.  
It performs basic CRUD (Create, Read, Update, Delete) operations using JDBC and MySQL, focusing on backend development fundamentals.

This project demonstrates database connectivity, object-oriented programming concepts, and clean application structure.

---

## Features
- Add new student records  
- View all student records  
- Search student by ID  
- Update student department  
- Delete student records  
- Menu-driven console interface

How to Run the Project
-Install Java (JDK 11 or above)
-Install and start MySQL Server
-Create the database and table using the SQL script
-Add MySQL Connector/J to the project
-Update database credentials in DatabaseUtil.java
-Run ApplicationRunner.java


Prerequisites
-Java installed
-MySQL running on port 3306
-MySQL Connector/J added to the project
